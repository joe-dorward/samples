Program (inches2.c)

Write a program that converts inches, to yards, feet, and inches

The program should:

  • Asks the user to enter a number of inches
  • Tells the user how many yards, feet, and inches make up the inches entered.
You should think about how you would do the calculation given the inches: 13, 25, 38, 50. Once you know how you would do it, you should be able to turn that into a calculation that the computer can do. See the example program output below.

Please enter a number of inches: 13
13 inches, converts to:
    0 yards.
    1 feet.
    1 inches.
-------------------------------------
Please enter a number of inches: 38
38 inches, converts to:
    1 yards.
    0 feet.
    2 inches.
-------------------------------------
Please enter a number of inches: 50
50 inches, converts to:
    1 yards.
    1 feet.
    2 inches.


Program (interest1.c)

Write a program  based on the variable declarations:

int deposit_term = 0,
    year_number = 0; // used as loop counter
    float deposited_sum = 0.0f,
    compounded_sum = 0.0f,
    annual_percentage_rate = 0.0f;

that will calculate the compounded interest on a sum of money deposited at a fixed rate of interest over a number of years.

The program should ask for:

  • The sum being deposited
  • The term of the deposit
  • The rate of interest to be applied annually.
The program should calculate, and list, the value of the deposited sum at the end of each year.

Program output:

Program (delete1.c)

Write a program that will delete an entry from a sorted array.  Use one of the states programs, and delete one of the entries.

  • Ask for the name of the state to delete from the array
  • Searches for that entry
  • If found, all the entries in the array above the one to be deleted get shuffled down (on top of the entry to be deleted)
  • Then the changed array has its contents printed out to the screen.

Program (insert1.c)

The oppisite of the last program, in this program you add an entry to a sorted array (in order).

  • Ask for the entry to add
  • Search for the place to insert the new entry
  • When found, shuffle rest of entries up
  • Insert new entry in position
  • Print contents of changed array

Program (unscramble1.c)

Write a program like scramble1.c, that will unscramble the string of letters. The program will ask for a scrambled string, and a key value, then print out the unscrambled string.


Program (unscramble2.c)

Write a program like unscramble1.c, that will NOT ask for a key value. Instead it will assume that the scrambled string has a key value between zero and nine, and it will print out an unscrabled string for each value. One of these unscrambled strings will be the correctly unscrambled version.
