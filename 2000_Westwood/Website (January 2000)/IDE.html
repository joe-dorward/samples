Driving the Visual C++ IDE (Integrated Development Environment)

If you dont already have a folder on the networked P:\> drive, create one now.

Creating the Source Code:

  1. Click the Windows Start button
  2. Go through: Programs>Microsoft Visual Studio 6.0, and click on Microsoft Visual C++ 6.0
  3. When the environment opens, click File, then New  a dialog box opens
  4. Click the Files tab
  5. Click C++ Source Files in the list
  6. In the File name: box, type the program file name: add_two.c
  7. Set the location to your storage area (perhaps your folder on the networked P:\> drive)
  8. Click the OK button
  9. Begin typing the program at the blinking insertion point (remember to save the file as you go along)

Once the program source code has been entered, you have to compile it.

If it compiles with no errors, you can run the program.

Most compiling errors will be simple typing errors, so look for these first.

Compiling the Source Code:
  1. Click Build, then Compile add_two.c  a dialog box opens asking you about a active project workspace
  2. Click the Yes button
  3. Click Build, then Execute add_two.c. This step completes the creation of the executable version of your program, and runs it.

You can re-run the program by clicking the red exclamation (!) mark above the edit window.

When you are finished with each program, you must close the project, and open a new one.


Closing a open Project:
  1. Clicking File, then Close Workspace  a dialog box opens asking you about closing all document windows
  2. Click the Yes button
Opening a new Project:
  1. Go back to step three in "Creating the Source Code:" above.
