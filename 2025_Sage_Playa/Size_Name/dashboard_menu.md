The **Dashboard** menu is used by organisational users to access the top-level dashboard pages that are appropriate for them to access. 

The **Dashboard** menu opens when organisational users hover their mouse pointer over it. 

The **Dashboard** menu is dynamic, responding to:
- The organisational type of the organisational user
- The role of the organisational user

#### Sage Support Agent
The **Dashboard** menu is shown to organisational users with the single **Sage Support Agent** role.

For **Sage Support Agent** organisational users, the **Dashboard** menu shows these options:
- **Operation Dashboard** - organisational users will click this option to open the **Operations** page
- **Site Dashboard** - organisational users will click this option to open the **Sites** page
- **BI Dashboard** - organisational users will click this option to open the **Business** page
